# 🧭 Wanderlust

**Wanderlust** is a full‑stack Airbnb‑style vacation rental platform built with **Node.js**, **Express**, **MongoDB**, and **EJS**.  
Users can explore travel stays, post listings with uploaded images, leave ratings & reviews, and manage their own listings — all with secure authentication and validation.

🚀 **Live Demo:** https://wanderlust-project-srxa.onrender.com

---

## 🛠️ Features

✨ Browse all travel listings  
✨ Create, edit & delete your own listings  
✨ Upload listing images via **Cloudinary**  
✨ Signup/Login with **Passport.js**  
✨ Leave star ratings & reviews  
✨ Flash message alerts for success/error  
✨ Responsive UI with **Bootstrap 5**  
✨ Server‑side validation using **Joi**

---

## 🧩 Tech Stack

| Layer | Technology |
|-------|------------|
| Backend | Node.js, Express.js |
| Database | MongoDB & Mongoose |
| Templating | EJS |
| Authentication | Passport.js (local strategy) |
| Uploads | Multer + Cloudinary |
| Styling | Bootstrap 5 |
| Validation | Joi |
